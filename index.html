Manage multiple Claude Code accounts from one terminal

Switch accounts, manage sessions, snapshot environments, and track usage — all without leaving your workflow.

Get Started View on GitHub
$ curl -fsSL https://raw.githubusercontent.com/dr5hn/ccm/main/install.sh | bash COPY
ccm — zsh
$ ccm list
Managed Accounts:
1 ★ darshan@personal.dev [personal] healthy
2 darshan@company.com [work] healthy
3 client@agency.io [client] healthy
$ ccm switch work
✓ Switched: personal → work
The Problem

Claude Code wasn't built for multiple accounts.

⚙
Manual config swaps
Editing ~/.claude/credentials every time you switch contexts
◎
Orphaned sessions
Stale project data silently eating your disk space
◻
No visibility
No way to see which account is active or how much space Claude uses

CCM fixes all of this with a single command.

See It In Action
ccm — zsh
What's Inside
⟩_
Account Switching
ccm switch <id>
Rotate between accounts in one command. Aliases, history, undo, import/export.
⊞
Session Management
ccm session list
List, inspect, relocate, and clean orphaned sessions eating your disk.
⊡
Environment Snapshots
ccm env snapshot
Save and restore your Claude Code config. Audit MCP servers for token waste.
⊟
Usage Stats
ccm usage summary
See your Claude Code footprint. Disk usage, session counts, project breakdown.
⊕
Health & Maintenance
ccm doctor
Scan for stale locks, log bloat, cache issues. Auto-fix with --fix. Targeted cleanup for logs, telemetry, cache.
⊘
Token Optimization
ccm optimize
Analyze context window footprint. Flag unused plugins and oversized CLAUDE.md files inflating token usage.
Get Started In 30 Seconds
1
Install
curl -fsSL .../install.sh | bash
One command, no sudo required. Works on macOS, Linux, WSL.
2
Add your accounts
ccm add
Run once per account. CCM detects your active Claude Code credentials.
3
Switch anytime
ccm switch work
Swap accounts instantly. Your sessions and config stay intact.
Install
$ curl -fsSL https://raw.githubusercontent.com/dr5hn/ccm/main/install.sh | bash
Requires: bash 4.4+, jq · Works on macOS, Linux, WSL
Or install as a skill: npx skills add dr5hn/ccm@ccm
Or clone manually: git clone https://github.com/dr5hn/ccm.git && cd ccm && ./install.sh
