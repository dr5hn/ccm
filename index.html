The power-user toolkit for Claude Code_

Manage accounts, sessions, environments, and usage — all from the terminal.

// live demo
ccm — bash
// modules
[>_]
Account Switching
ccm switch <id>
Rotate between Claude Code accounts with a single command. Aliases, history, undo, import/export.
[>>]
Session Management
ccm session list
List, inspect, relocate, and clean up project sessions. Find orphaned data eating your disk.
[:::]
Environment Snapshots
ccm env snapshot
Save and restore your Claude Code configuration. Audit MCP servers for token efficiency.
[|||]
Usage Stats
ccm usage summary
See your Claude Code footprint. Top projects by disk usage, session counts, orphaned data.
// install
$ curl -fsSL https://raw.githubusercontent.com/dr5hn/ccm/main/ccm.sh -o ccm.sh && chmod +x ccm.sh
Requires: bash 4.4+, jq · Works on macOS, Linux, WSL
// command reference
# Account Management
ccm addAdd current account
ccm remove <id>Remove account by number, email, or alias
ccm switch [id]Switch account (next or specific)
ccm undoRevert last switch
ccm listList managed accounts
ccm statusActive account details
ccm alias <id> <name>Set account alias
ccm verify [id]Verify backup integrity
ccm historyShow switch history
ccm export <path>Export accounts to archive
ccm import <path>Import from archive
ccm interactiveInteractive menu mode
# Session Management
ccm session listList all project sessions
ccm session info <path>Session details for a project
ccm session relocate <old> <new>Move sessions after project relocation
ccm session clean [--dry-run]Remove orphaned sessions
# Environment
ccm env snapshot [name]Save current environment
ccm env restore <name> [--force]Restore from snapshot
ccm env listList snapshots
ccm env delete <name>Remove a snapshot
ccm env auditAudit MCP token efficiency
# Usage Stats
ccm usage summaryClaude Code footprint overview
ccm usage top [--count N]Top projects by disk usage
